RulesetEntity.php : 14

To parse ruleset from string to entity.

Ruleset is the set of rules(methods) of a data and its children. Ruleset is finally defined by using simple string. We have to parse it every time when we start validating the rules. We parse the ruleset into the ruleset entity. For the same ruleset, there is no need to parse it again the next time it is validated.
